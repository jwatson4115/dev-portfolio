Building A Full-Scale Music Streaming Site

An upcoming music streaming platform built with Angular, ASP.NET Core, Redux, and the YouTube API.


The website is built upon an innovative new approach to playlist management known as tags. Tags allow you to make very complex playlists that give full control over shuffling algorithms. Instead of being limited to shuffling songs, you can shuffle by albums, artists, and genres all mixed in the same playlist.

Achilles Rupture Recovery Application

A mobile application which aids patients through the recovery process of an achilles tendon rupture, similar to a fitness application. Currently being built as a team of four developers, with myself as lead front-end developer, in association with the NHS.


The cross-platform mobile application is being built using the Ionic framework, with ASP.NET as our back-end framework of choice.
