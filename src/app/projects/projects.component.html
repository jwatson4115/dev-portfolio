Combining Music Streaming Services

Winner of best student project 2016 as voted by school students.

A single-page application which can interweave music streaming results from a range of sites such as YouTube and SoundCloud. The site can autoplay flawlessly from one streaming site to the next within the same central playlist. Primarily built using AngularJS and JavaScript on the front-end, along with Bootstrap for mobile-first styling.
